The Box Model

In this post I want to explain the Box Model as it refers to Web Design and the difference between Padding, Borders and Margin.

24 November 2019

In HTML we can view elements as boxes. The way we apply style to those elements is through CSS and what is known as ‘The Box Model’. We can think of the Box Model as a box that wraps around every HTML element.

This box contains four different parts; Content, Padding, Borders and Margins. (See image below)

  • Content - The content is the centre of the box, this is where all of our text and images are stored.
  • Padding - This is the area around the content, this clears the area away from the content.
  • Border - Creates a border around the padding and the content
  • Margin - Clears the area around the border. It is the ‘space between boxes’

A good analogy is to think of the element in terms of a shoebox. Imagine you are looking down onto an open shoebox, the shoes would be the content, the wrapping paper inside the box would be the padding and the edge of the box would be the border. Now imagine you place another shoe box next to this one. The space between the two boxes would be margin.

One important point to remember when designing websites is that when you set the height and width values of an element this only relates to the content area. In order to calculate the full size of the element you must also include any padding, borders and the margins.
